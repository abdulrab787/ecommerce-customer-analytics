# Power BI — Ecommerce Analytics

This folder contains the main Power BI solution for the Ecommerce Analytics project.

- `dashboards/Ecommerce_Analytics.pbix` — 5-page report:
  - Executive Overview
  - RFM Segmentation
  - Churn & Retention
  - Profitability
  - Channel & Audience Insights

- `models/` — Data model diagrams (to be added)
- `measures/` — DAX measures organized by logical folders
- `assets/` — Screenshots used in documentation/portfolio
